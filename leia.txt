estrutura esperada para criar manualmente:

meu_projeto/
│── docker-compose.yml
│── src/                    # pasta vazia para receber o código Laravel
│── php/
│   └── Dockerfile
│── nginx/                  # configs do Nginx
│   └── default.conf
|── .env                    # para dados do banco de dados e outras coisas que você quiser

para gerenciar o projeto laravel quando estiver criado:
docker compose exec meu_projeto_app <comando desejado>

consulta amigável com nome do container:
docker ps --format "table {{.ID}}\t{{.Names}}\t{{.Image}}\t{{.Status}}"

Configuração do vite para docker (vite.config.js):
server: {
    host: "0.0.0.0",
    port: 5173, // porta padrão do vite
    strictPort: true,
    hmr: {
        host: "ip.do.seu.host", // ou nome do host, mas sempre sem a porta
        port: 5173 // porta padrão do vite
    }
}

.env do laravel (src/.env):
APP_URL=http://<ip do seu host>:<porta do container php>
DB_ROOT_PASSWORD=senha do root definida no .env raiz
